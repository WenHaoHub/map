# 世界地图使用说明

## 功能概述

基于现有的中国地图组件，新增了世界地图功能，支持两级下钻：世界地图 → 单个国家地图。

## 主要功能

### 1. 地图切换
- 在页面顶部有两个切换按钮：「中国地图」和「世界地图」
- 可以在中国地图和世界地图之间自由切换

### 2. 世界地图功能
- **世界地图展示**：显示全球所有国家的边界
- **国家下钻**：点击任意国家可查看该国家的详细地图
- **面包屑导航**：支持从国家地图返回世界地图
- **交互功能**：支持地图缩放、平移等操作

## 技术实现

### 数据源
- **世界地图数据**：`/worldGeoJSON/countries.geo.json`
- **单个国家数据**：`/worldGeoJSON/GeoJSON/{国家代码}.geo.json`
- 使用标准的GeoJSON格式
- 国家代码采用ISO 3166-1 alpha-3标准

### 组件结构
```
WorldMap.vue
├── 顶部导航栏（面包屑 + 提示信息）
├── 地图容器（ECharts渲染）
└── 加载状态提示
```

### 交互流程
1. **初始加载**：加载世界地图数据并渲染
2. **点击国家**：获取国家代码，加载对应的国家GeoJSON文件
3. **面包屑导航**：点击"世界"可返回世界地图

## 支持的国家

目前支持约177个国家和地区，包括：
- 主要大国：中国(CHN)、美国(USA)、俄罗斯(RUS)、印度(IND)等
- 欧洲国家：德国(DEU)、法国(FRA)、英国(GBR)、意大利(ITA)等
- 亚洲国家：日本(JPN)、韩国(KOR)、泰国(THA)、越南(VNM)等
- 其他各大洲国家

## 使用方式

1. **启动项目**：
   ```bash
   npm run dev
   ```

2. **切换到世界地图**：
   - 点击页面顶部的「世界地图」按钮

3. **查看国家详情**：
   - 在世界地图上点击任意国家
   - 系统会加载该国家的详细地图

4. **返回世界地图**：
   - 点击面包屑导航中的「世界」

## 特色功能

### 1. 响应式设计
- 支持窗口大小变化
- 地图自动调整尺寸
- 移动端友好的交互体验

### 2. 国家名称本地化
- 支持中文国家名称显示
- 覆盖177个国家和地区
- 未覆盖的国家显示英文名称

### 3. 交互体验
- 鼠标悬停显示国家名称
- 点击高亮效果
- 平滑的缩放和平移操作

## 扩展说明

如需添加更多国家或修改功能：

1. **添加国家GeoJSON**：
   - 在`/worldGeoJSON/GeoJSON/`目录下添加对应的`.geo.json`文件
   - 文件名使用ISO 3166-1 alpha-3国家代码

2. **添加中文名称**：
   - 在`WorldMap.vue`的`countryNames`对象中添加映射关系

## 注意事项

1. **数据文件路径**：确保GeoJSON文件放在正确的public目录下
2. **国家代码标准**：使用ISO 3166-1 alpha-3标准（3位字母代码）
3. **文件大小**：某些大国的GeoJSON文件较大，加载可能需要时间
4. **浏览器兼容性**：建议使用现代浏览器以获得最佳体验

## 总结

世界地图功能完全满足了使用ECharts做世界地图查看并实现下钻到国家的需求：

✅ **世界地图展示**：使用`countries.geo.json`显示全球地图  
✅ **国家下钻**：点击国家后加载单个国家的GeoJSON文件  
✅ **单国家展示**：下钻后只显示单个国家，不显示省州  
✅ **交互体验**：完整的导航、缩放、悬停等交互功能  

现有的`worldGeoJSON`资源完全能够支持这个功能的实现。
